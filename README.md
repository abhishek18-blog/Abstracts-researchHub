## 🌟 About the Project

Abstracts is a state-of-the-art Research Dashboard built with modern web technologies. It leverages AI-augmented workflows to streamline research activities, making it an indispensable tool for academia. 

This project demonstrates strong full-stack capabilities, modern UI/UX sensibilities, and complex API integrations.

### ✨ Key Features

- **🔍 Intelligent Paper Discovery**: Search millions of papers via the Semantic Scholar API integration.
- **📚 Dynamic Research Library**: Organize findings into tailored projects with progress tracking.
- **🤖 Built-in AI Assistant**: Summarize abstracts, explain complex concepts, and generate paper suggestions via an integrated AI chat interface.
- **🤝 Community Hub**: Dedicated spaces for peer review, discussions, and shared insights.
- **🌗 Theme-Aware UI**: Beautifully crafted dark/light modes featuring glassmorphism, dynamic gradients (`#1e1b4b` to `#3b82f6`), and micro-animations.

---

## 🎨 Design & Architecture

Abstracts is designed with a **Premium Modern SaaS** aesthetic inspired by the original [Figma Design](https://www.figma.com/design/LhqME4tTeWU7HZPIsjkn62/Design-ResearchHub-Dashboard):
- **Color Palette**: Sophisticated dark themes paired with deep blues (`#1e40af`, `#3b82f6`, `#030213`) and elegant typography.
- **Micro-Interactions**: Powered by modern CSS animations for a fluid, premium user experience.
- **System Workflow**: React/Vite Frontend ↔ Express Server ↔ MongoDB Atlas.

## 🚀 Getting Started

Follow these steps to run the application locally.

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
# 1. Clone the repository
git clone https://github.com/abhishek18-blog/Abstracts-researchHub.git

# 2. Install dependencies
npm i

# 3. Start the development server
npm run dev
```

> **Note**: A companion backend server may be required to resolve API dependencies. Ensure environment variables (`.env`) are securely configured.
